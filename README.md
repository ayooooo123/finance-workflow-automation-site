# Finance Workflow Automation Site

Static HTML/CSS/JS proposal demo for an Upwork listing: website design + build for a finance workflow automation firm.

## Structure

- `index.html` — layout and sections
- `content.js` — editable copy/content config
- `styles.css` — responsive minimalist visual system
- `script.js` — content rendering, reveal animation, demo form note
- `_headers` — Cloudflare Pages security/cache headers

## Local check

```bash
npm run check
```

## Deployment

This is designed for Cloudflare Pages. No build step is required; deploy the repo root.
